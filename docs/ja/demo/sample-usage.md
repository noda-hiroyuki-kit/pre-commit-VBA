# 使用方法のデモ

このデモは `pre-commit-vba` の導入手順です.  
初学者でも再現できる順に進めます.

## このデモでできること

- GitHub にリポジトリを作れます.
- Git Flow の基本運用を試せます.
- `pre-commit-vba` の導入手順を体験できます.
- リリースまでの流れを確認できます.

## 事前準備

次のツールが使える状態にします.

- `git`
- `mise` または `uv`
- GitHub アカウント
- Excel（マクロ有効ブックを扱える環境）

## 進め方

各ステップは次の順で読んでください.

1. 目的を確認します.
2. 手順をそのまま実行します.
3. 確認ポイントで成功を判定します.
4. 問題があれば補足を確認します.
