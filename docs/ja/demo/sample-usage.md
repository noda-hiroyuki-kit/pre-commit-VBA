# 使用方法のデモ

この-pre-commit-vbaの設定方法を[pre-commit-vba-example](https://github.com/noda-hiroyuki-kit/pre-commit-vba-example)の作成を通じて説明する.

- ブランチ戦略はgit flowに従う.
- 作業ディレクトリで`git`, `mise`を利用できることが必要.

5.  [Excelのアプリを構築](step-05/create-excel-app.md)
